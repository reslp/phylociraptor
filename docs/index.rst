Phylociraptor is a bioinformatics pipeline to create phylogenomic trees. It is easy to install, scalable, reproducable and fast and performs all steps of a typical phylogenomic analysis.


----------------------
Getting started
----------------------

* Read what phylociraptor can `do <introduction/about.html>`_ . 
* Learn how to obtain and install phylociraptor `here <introduction/installation.html>`_ .
* You can read the `quickstart <introduction/quickstart.html>`_ page to see how phylociraptor is run.
* To learn all about what phylociraptor does you can read your `in-depth <indepth/setup.html>`_ guide.


----------------------
Getting Help
---------------------- 

If you run into problems with phylociraptor please follow these steps:

1. Review the relevant parts of the manual.
2. Check the `known-issues <getting-help/problems.html>`_ page if your problems is already known.
3. Search the `issue tracker <https://github.com/reslp/phylociraptor/issues>`_ if other people are having the same problem.
4. Post an `issue <https://github.com/reslp/phylociraptor/issues>`_  on GitHub describing your problem.
